System Requirements: Python 3 (modules needed: flask, docxtpl, docx, matplotlib, pandas, csv), MS Word

0. Edit \ReportsGenerator\config.csv;
1. Start cmd window, direct to \ReportsGenerator;
2. Type "pip install -r requirments.txt" to install modules;
3. After successfully installed all modules, type "python ReportsGenarator1.0.py" to start the flask server;
4. Open "localhost:5000" in a web browser to start generate reports;
